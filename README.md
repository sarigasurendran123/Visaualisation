# Visualization Projects

This repository showcases my data visualization projects built using Excel, Power BI, and Tableau.
These projects are designed to present data in a clear, interactive, and meaningful way for better insights and decision-making.

## Projects Included

1. Inventory Management Dashboard
Tool Used: Power BI  
This dashboard helps analyze inventory-related data such as stock levels, product movement, and overall inventory performance.

 2. Students Performance Dashboard
Tool Used: Tableau  
This dashboard visualizes student performance data and helps understand academic trends, scores, and comparisons.

 3. Supermarket Sales Dashboard
Tool Used: Excel  
This dashboard provides insights into supermarket sales performance, including revenue, category-wise analysis, and sales trends.

## Tools & Technologies
- Microsoft Excel
- Power BI
- Tableau
